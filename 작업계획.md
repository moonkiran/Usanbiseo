# 우산비서(Umbrella Assistant) 작업 계획서

본 계획서는 '우산비서' 프로젝트의 1차 고도화 작업 완료 사항과 향후 시스템 안정성 및 사용자 경험 향상을 위해 진행해야 할 개발 로드맵을 정리한 문서입니다.

---

## 1. 지금까지 진행된 과정 (Completed Tasks)

### 1) 사용자 경험(UX) 및 인터랙션 최적화
* **스플래시 화면 전환 속도 개선**
  * 대기 시간을 기존 2초에서 **0.8초**로 대폭 단축하고 페이드아웃 효과(0.4초)를 적용하여 앱 진입 시 경쾌한 느낌을 주도록 개선했습니다.
* **하단 메인 버튼 탄성 팝업 (Staggered Pop-up)**
  * 메인 화면 노출 시 로그인 버튼(0.1s 딜레이)과 회원가입 버튼(0.22s 딜레이)이 차례대로 아래에서 위로 튕기며 나타나는 스프링 탄성 효과(`cubic-bezier(0.34, 1.56, 0.64, 1)`)를 구현했습니다.
* **대시보드 페이지 요소 로딩 인터랙션 적용**
  * 대시보드 진입 시 상단바 ➡️ 캐릭터 ➡️ 현재 날씨 ➡️ 상세 정보 카드 ➡️ 하단 메뉴바 순으로 0.1초 간격의 시차를 두고 탄력적으로 팝업되는 인터랙션을 반영했습니다.
* **대시보드 세부 레이아웃 조정**
  * 캐릭터(`.character`)의 세로 위치를 **20% 더 아래로** 하강시키고, 말풍선(`.speech-bubble`)을 **오른쪽으로 10% 이동**시켜 시각적인 안정감과 밀착감을 강화했습니다.

### 2) 신규 온보딩 흐름 개발
* **가입 환영 페이지 (`welcome.html`) 신규 제작**
  * 참고 디자인 시안의 소프트 파스텔 블루 배경, 오리 캐릭터 및 그림자 레이아웃을 100% 매칭하여 구현했습니다.
* **가입 플로우 기반 동적 라우팅 구현**
  * 랜딩페이지(`index.html`)에서 '회원가입' 버튼을 통해 가입을 완료하고 첫 로그인을 성공한 경우에만 `welcome.html` 페이지로 이동하고, 일반 로그인 시에는 대시보드로 바로 진입하는 상태 플로우(`fromSignup` localStorage 플래그)를 연동했습니다.
* **말풍선 타이핑 애니메이션 및 스킵 로직**
  * 우비 캐릭터의 말풍선 내부에서 타이핑을 치는 듯한 모션을 정교하게 구현했으며, 타이핑 완료 2.5초 후 자동 대시보드 이동 기능과 화면 터치 시 즉시 스킵되는 UX 편의 기능을 추가했습니다.

### 3) 웹 표준 및 SEO 최적화
* **검색엔진 최적화(SEO) 및 웹 접근성 반영**
  * `welcome.html` 파일에 명확한 Meta Description을 설정하고, 구조적 웹 표준에 맞는 단일 `<h1>` 요소(비주얼 숨김 처리)를 배치했습니다.
  * 자동화 테스트 및 유지보수를 고려하여 주요 상호작용 요소에 고유한 ID(`welcomeAppContainer`, `skipGuide`)를 추가했습니다.
* **코드 스타일 일관성 확립**
  * 프로젝트 전반의 코드 스타일 일치화를 위해 환영 페이지(`welcome.html`) 내부의 `<style>` 태그를 body 최하단 영역(스크립트 아래)으로 재배치했습니다.
* **회원가입 폼 직관성 개선**
  * `signup.html`의 폼 하단 제출 버튼 텍스트가 기존 '로그인'으로 잘못 노출되던 문제를 '회원가입 완료'로 바로잡고, 가입 성공 알림창 피드백을 추가했습니다.

## 2. 앞으로 더 해야 할 과정 (Future Roadmap)

### 1) 데이터 모델 및 API 연동
* **실시간 날씨 데이터 연동 (OpenWeatherMap API)**
  * 현재 하드코딩되어 있는 대시보드의 날씨, 강수 확률, 온도 값 등을 실제 기상청 또는 OpenWeatherMap Open API와 연동하여 실시간으로 렌더링되도록 구현합니다.
* **위치 정보 자동 획득 (Geolocation API)**
  * 사용자가 위치 권한을 동의했을 때 브라우저의 `navigator.geolocation` API를 사용해 실제 현재 위도·경도 값을 획득하고, 역지오코딩(Reverse Geocoding)을 통해 사용자의 실제 주소 명칭("서울특별시 강남구" 등)을 대시보드 상단에 자동 주입합니다.

### 2) 주요 기능 구체화 및 스크립트 연결
* **일정 관리 기능 (CRUD & LocalStorage 연동)**
  * `schedule.html`에서 제공되는 일정 추가/삭제/체크 버튼 클릭 시 모달창에서 입력받은 데이터가 LocalStorage에 저장·유지되어, 새로고침 시에도 일정이 보존되고 대시보드 요약 정보에 반영되도록 자바스크립트 비즈니스 로직을 구축합니다.
* **우비톡 AI 인터랙션 및 시나리오 다변화**
  * `wubitalk.html`에서 사용자가 입력창에 직접 텍스트를 전송하거나 AI 추천 질문 버튼을 눌렀을 때, 우비 오리 비서가 날씨 상황 및 일정 기반의 맞춤 답변을 실시간 타이핑 애니메이션과 함께 대답하는 대화형 엔진을 개발합니다.
* **개인화 설정 반영 (설정값 활성화)**
  * `settings.html`의 알림 시간 설정, 특정 알림 여부 토글 값 등을 설정 시 LocalStorage에 보존하고, 이에 맞춰 대시보드 캐릭터 멘트와 알림 스케줄러가 변동되도록 결합합니다.

### 3) 성능 최적화 및 안정성 보완
* **리소스 및 애니메이션 최적화**
  * 배경에 적용된 빗방울 낙하 애니메이션 및 스크롤바 관성 효과가 저사양 기기나 모바일 환경에서 끊김 없이 60fps로 매끄럽게 동작하도록 CSS 하드웨어 가속(`will-change` 속성)을 적용합니다.
* **반응형 뷰포트 크로스브라우징 검증**
  * 다양한 모바일 기기(아이폰, 갤럭시 등)의 노치 디자인 및 웹뷰 환경에서 깨짐 없이 390px 앱 쉘 템플릿이 화면 중앙에 올바르게 고정되고 렌더링되는지 크로스브라우징 테스트를 수행합니다.
